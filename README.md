# LocoLab

Umbrella project for the Loco research lab, exploring local AI on consumer hardware.

## Projects

| Project | Description |
|---------|-------------|
| [loco-bench](https://github.com/michael-borck/loco-bench) | Systematic benchmarks of quantized small language models on consumer hardware |
| [loco-convoy](https://github.com/michael-borck/loco-convoy) | Documentation and experiments for running AI inference workloads across multiple GPUs |
| [loco-llm](https://github.com/michael-borck/loco-llm) | Builds a collaborative ecosystem of task-specific fine-tuned small language models that run on consumer hardware |
| [loco-ensayo](https://github.com/michael-borck/loco-ensayo) | Rehearsal platform for AI-driven teaching scenarios |
